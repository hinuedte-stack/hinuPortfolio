# Portfolio Website - Hafiza Hydar Hinu

## Project Title
**Portfolio Website - Hafiza Hydar Hinu**

## Project Description
A modern, responsive portfolio website showcasing skills, projects, achievements, and contact information. This portfolio features a professional hero section, interactive skills display with animated progress bars, project gallery with video demos, achievements section, and a fully functional contact form with validation. The website is built with a mobile-first approach and includes smooth scrolling navigation, hover animations, and responsive design for optimal viewing across all devices.

## Technologies Used
- **HTML5** - Semantic markup (header, nav, main, section, footer)
- **CSS3** - Modern styling with gradients, animations, and responsive design
- **JavaScript** - Interactive elements, form validation, smooth scrolling
